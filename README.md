# INTEL 4004 Emulator
#### This project is an independent emulator of the Intel 4004 microprocessor and is not affiliated with, sponsored by, or endorsed by Intel Corporation.

## Introduction
Intel 4004 is the first microprocessor developed by Intel and it is a 4-bit processor.<br>
Intresting this is is uses completely different architecture and later upgrades where made to bring out X86 processor which is a standard architecture used today in many PC.<br>
This is a Emulator which tries to emulate the legendary 4004 processor.
